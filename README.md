# github-cheatsheet
commands for using github
1. Create branch
```console
$ git checkout -b <branch_name>
```
2. Make edits.
3. stage edits.
   ```console
   git add .
   ```
4. make commit.
   ```console
   $ git commit -m "commit message"
   ```
 5. push commit
 ```console
 git push origin branch_name
 ```
 6. create PR
```console
 gh pr create
```

7. submit.
```console
 gh pr merge
```
